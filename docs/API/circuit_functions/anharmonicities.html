Calculating anharmonicities¶

Qcircuit.anharmonicities(**kwargs)[source]¶

Returns the anharmonicity of the circuit normal modes.

The array is ordered ordered with increasing normal mode frequencies. Such that the first element of the array corresponds to the loss rate of the lowest frequency mode. Losses are provided in units of Hertz, not in angular frequency.

Parameters:kwargs – Values for un-specified circuit compoenents, ex: L=1e-9.
Returns:Normal mode anharmonicities
Return type:numpy.array

Notes

The Hamiltonian of the circuit is

\(\hat{H} = \sum_m hf_m\hat{a}_m^\dagger\hat{a}_m + \sum_j E_j[1-\cos{\hat{\varphi_j}}-\frac{\hat{\varphi_j}^2}{2}]\),

where \(\hat{a}_m\) is the annihilation operator of the m-th normal mode of the circuit and \(f_m\) is the frequency of the m-th normal mode, \(E_j\) is the Josephson energy of the j-th junction and

\(\varphi_j = \sum_m\frac{\phi_{zpf,m,j}}{\phi_0}(\hat{a}_m^\dagger+\hat{a}_m)\).

where \(phi_0 = \hbar/2e\) and

\(\phi_{zpf,m} = \sqrt{\frac{\hbar}{\omega_mImY'(\omega_m)}}\)

is the zero-point fluctuations in flux if a mode through the junction, with frequency \(\omega_m\) and admittance to the rest of the circuit \(Y\)

By keeping only terms which play a role up to first order perturbation

\(\hat{H} = \sum_m\sum_{n\ne m} h(f_m-A_m-\frac{\chi_{mn}}{2})\hat{a}_m^\dagger\hat{a}_m -h\frac{A_m}{2}\hat{a}_m^\dagger\hat{a}_m^\dagger\hat{a}_m\hat{a}_m -h\chi_{mn}\hat{a}_m^\dagger\hat{a}_m\hat{a}_n^\dagger\hat{a}_n\)

This function returns the anharmonicities

\(A_m = \sum_j A_{m,j}\)

where

\(A_{m,j} = E_j/2/h\left(\frac{\phi_{zpf,m,j}}{\phi_0}\right)^4\)

is the contribution of junction j to the total anharmonicity of a mode m.
